## 介绍

`bbs-go`是一个使用Go语言搭建的开源社区系统，采用前后端分离技术，Go语言提供api进行数据支撑，用户界面使用Nuxt.js进行渲染，后台界面基于element-ui。如果你正在学习Go语言，或者考虑转Go语言的Phper/Javaer...那么该项目对你有的学习会有很大的帮助，欢迎一起来交流。

## 模块

### server

> 基于`Golang`搭建，提供接口数据支撑。

### site

> 前端页面服务，基于`nuxt.js`搭建。
